owatchSendMessage(3)
====================
Victor Semizarov <vitkinet@gmail.com>

NAME
----
owatchSendMessage - ...


SYNOPSIS
--------
*#include <optikus/watch.h>*

*oret_t owatchSendMessage (void $$*$$'data', int 'nbytes')*;


DESCRIPTION
-----------
This routine sends the data packet from the buffer pointed by 'data' of the
'nbytes' length.
